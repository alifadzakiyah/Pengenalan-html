Selamat datang

Matakuliah desain web ini bertujuan untuk membina keahlian dalam bidang Desain Web, mulai dari menjelaskan berbagai istilah teknis terkait pengembangan website, menjelaskan potensi website pembelajaran sesuai kebutuhan lembaga/institusi, mengidentifikasi berbagai platforms content management systems (CMS), merancang rencana pengembangan dan mengembangankan website pembelajaran berbasis CMS.

Web 1.0
Web 1.0 merupakan web generasi pertama yang menurut Berners-Lee dapat dianggap sebagai web hanya-baca.
Web 2.0
Web 2.0 menurut Dale Dougherty adalah web baca-tulis yang menjadi media komunikasi antara manusia.
Web 3.0
Web 3.0 dikenal sebagai web semantik, yaitu web yang dapat dibaca oleh mesin, tidak hanya oleh manusia.
